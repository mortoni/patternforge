## Typography

- **UI:** Inter / system sans (`--font-inter`)
- **Docs:** Slightly relaxed line height (`leading-[1.7]`) for long-form readability
- **Code:** Geist Mono for commands and schema snippets

## Spacing & shape

- **Radius:** `rounded-lg` / `rounded-xl` for cards and callouts
- **Docs rhythm:** Section gaps ~`mt-14`, paragraph `mb-5` — see `src/mdx-components.tsx`
- **Training panels:** Compact vertical padding; board gets visual priority

## Components

Built on shadcn/ui patterns in `src/components/ui/`. Variants use `class-variance-authority` — extend variants rather than one-off classes when a pattern repeats.
