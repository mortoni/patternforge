# Data Model

**Why this exists:** Quick reference for persisted entities. **`src/db/schema.ts`** is authoritative — types are Zod-inferred.

## Training set

```typescript
interface TrainingSet {
  id: string;
  name: string;
  difficulty: "easy" | "intermediate" | "advanced" | "custom";
  exerciseIds: string[];
  createdAt: string;
  description?: string;
  source?: string;
  tags?: string[];
}
```

Set size = `exerciseIds.length`.

## Exercise

```typescript
interface Exercise {
  id: string;
  trainingSetId: string;
  fen: string;
  sideToMove: "w" | "b";
  solutionMoves: string[]; // UCI after import
  firstMove?: string;      // expected first move (UCI)
  puzzleNumber?: number;
  motifTags?: string[];
  comment?: string;
}
```

Training compares your move to **`firstMove`** / first solution move — not the full line.

## Cycle run

```typescript
interface CycleRun {
  id: string;
  trainingSetId: string;
  cycleNumber: number;
  status: "active" | "completed";
  totalTimeMs: number;
  solvedCount: number;
  totalExercises: number;
  nextExerciseIndex: number;
  startedAt: string;
  completedAt?: string;
}
```

## Session

```typescript
interface Session {
  id: string;
  trainingSetId: string;
  cycleRunId: string;
  status: "active" | "completed" | "abandoned";
  activeTimeMs: number;
  puzzlesAttempted: number;
  correctCount: number;
  skippedCount: number;
  startedAt: string;
  endedAt?: string;
}
```

## Exercise attempt

Table: **`exerciseAttempts`**

```typescript
interface ExerciseAttempt {
  id: string;
  exerciseId: string;
  cycleRunId: string;
  sessionId?: string;
  result: "correct" | "incorrect" | "skipped";
  durationMs: number;
  userMoves: string[];
}
```

## Mistake entry

Table: **`mistakeEntries`**

```typescript
interface MistakeEntry {
  id: string;
  exerciseId: string;
  trainingSetId: string;
  status: "needs_review" | "solved_once" | "solved_twice" | "mastered";
  failedAttempts: number;
  solvedReviewCount: number;
}
```

## Relationships

- Set → exercises via `exerciseIds`
- Set → many cycle runs
- Cycle run → many sessions → many attempts
- Mistake entry → one exercise + one set

App metadata: **`appInstance`**, **`settings`** (theme, board, last set).

See [State & Persistence](/docs/architecture/persistence) for the storage overview.
