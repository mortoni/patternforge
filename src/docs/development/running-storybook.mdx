# Running Storybook

**Why this exists:** Storybook is PatternForge's curated docs surface — UI review, product handbook, and developer guides.

## What it covers

- **Product & architecture** — training philosophy and system design
- **Development guides** — setup, tests, conventions
- **Puzzle Inspector** — internal puzzle validation tool

## Run locally

```bash
pnpm storybook
```

Opens at [http://localhost:6006](http://localhost:6006).

Static build (Chromatic CI):

```bash
pnpm build-storybook
```

Output: `storybook-static/`.
