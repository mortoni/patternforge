## Flow (simplified)

```
Training Set
    └── exercises (ordered)
    └── cycleRuns
            └── sessions
                    └── exerciseAttempts → exercise
            └── exerciseAttempts (may omit sessionId)
    └── mistakeEntries → exercise
```

## Canonical types

Use **Zod-inferred types from `schema.ts`** as the source of truth for persisted rows. The `src/domain/` folder holds complementary naming and import schemas — not a second persistence layer.

## Writes worth understanding

- **Seed** — `ensureGeneratedPuzzlesInDbIfEmpty()` loads JSON bundles when empty
- **Start cycle** — creates `cycleRun`, may open `session`
- **Solve / skip** — appends `exerciseAttempt`, advances cycle progress
- **Complete cycle** — marks run completed, enables reflection data

Full field reference: [Data Model](/docs/data-model).
