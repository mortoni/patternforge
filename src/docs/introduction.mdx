# Introduction

PatternForge is a chess tactics training application built around structured repetition and pattern recognition. This documentation explains the philosophy, method, and architecture behind the system.

## What is PatternForge?

PatternForge helps chess players build fast, reliable pattern recognition through disciplined repetition inspired by the Woodpecker Method. Unlike puzzle apps that encourage endless novelty, PatternForge focuses on returning to the same ideas until recognition becomes automatic.

## Core Principles

- **Deliberate repetition** over novelty
- **Pattern memory** over calculation support
- **Structured cycles** over random practice
- **Real board vision** over puzzle streaks
