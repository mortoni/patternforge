# PatternForge

Train patterns. Build recognition.

PatternForge is a **local-first tactical chess training system** built around repetition cycles and pattern recognition. This handbook covers the product, architecture, and how to work on the codebase.

<DocCallout title="Why this handbook exists">
  Storybook is the curated documentation surface for PatternForge — product intent, system design,
  UI review, and contributor guides in one place. The README stays short; details live here.
</DocCallout>
