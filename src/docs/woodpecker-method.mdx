# The Woodpecker Method

**Why this exists:** PatternForge adapts a well-known repetition method — fixed positions, repeated cycles, decreasing total time — into a structured local-first training flow.

## The method

1. **Choose a set** — Curated exercises sized to your level and schedule
2. **Complete cycle 1** — Solve the full set carefully
3. **Rest briefly** — Reset before the next pass
4. **Repeat faster** — Same positions, less total time, same quality bar
5. **Build automaticity** — Motifs surface with less friction each cycle

## The point is recognition

Not novelty. Recognition.

Returning to the same positions shifts effort from calculation to pattern memory. What once required analysis becomes immediate sight.
